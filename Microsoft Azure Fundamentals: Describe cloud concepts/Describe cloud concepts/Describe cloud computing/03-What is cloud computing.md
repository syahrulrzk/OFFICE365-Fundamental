# What is cloud computing

Sama seperti saat Anda berbelanja untuk komputer Anda sendiri, komputasi cloud memungkinkan Anda memilih kemampuan dan fitur yang Anda butuhkan untuk menjalankan perangkat lunak Anda.
Perbedaannya adalah dengan komputasi cloud, PC berada di pusat data penyedia cloud, bukan secara fisik bersama Anda.
Ini memungkinkan Anda hanya membayar layanan yang Anda gunakan.
Selain itu, orang lain dapat mengelola pemeliharaan komputer tersebut.

Setiap penyedia cloud akan memiliki pilihan layanannya sendiri untuk dipilih, tetapi layanan dasar yang disediakan oleh semua penyedia cloud adalah daya komputasi dan penyimpanan.
CDaya komputasi adalah seberapa banyak pemrosesan yang dapat dilakukan komputer Anda.

Misalnya, saat membeli komputer untuk di rumah, Anda dapat memilih komputer dengan RAM delapan gigabita dan prosesor terbaru untuk menjalankan perangkat lunak yang Anda butuhkan saat ini.
Namun seiring bertambahnya beban pada komputer, Anda akan merasakan bahwa komputer tersebut melambat.

Dengan komputasi cloud, Anda dapat menambah dan menghapus daya komputasi sesuai kebutuhan.
Ini dapat menghemat biaya, karena Anda hanya membayar sumber daya yang Anda gunakan.

Penyimpanan adalah volume data yang dapat Anda simpan di komputer Anda.
Komputer biasa memiliki ruang hard drive yang terbatas.
Seiring waktu, Anda dapat kehabisan penyimpanan, dan membeli hard drive lain untuk menyimpan lebih banyak data.
Dengan komputasi cloud, Anda dapat meminta lebih banyak penyimpanan saat Anda membutuhkannya.

Penyedia cloud juga mengelola pemeliharaan komputer sehingga Anda tidak perlu melakukannya.
Penyedia akan memastikan bahwa tersedia cadangan, bahwa sistem operasi selalu tarbarui, serta memastikan bahwa semuanya aktif dan berjalan selama 24 jam dalam sehari.
Jadi, seiring pertumbuhan bisnis Anda dan kebutuhan komputasi Anda berubah, Anda dapat dengan cepat menyediakan sumber daya komputasi baru dengan cara yang hemat biaya.
